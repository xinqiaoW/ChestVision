## 五、功能-数据库映射速查

| 功能 | 核心表 |
|------|--------|
| 用户注册（含类型） | `users` |
| 病人档案 | `patient_profiles` |
| 医患关系 | `doctor_patient_relations` |
| 病例管理 | `medical_records` + `medical_record_attachments` |
| 胸片检测 | `detection_scenes` → `detection_tasks` → `detection_results` |
| 胸片记录 | `cxr_images` |
| 检测报告 | `detection_reports` |
| 对话交互 | `chat_sessions` → `chat_messages` → `chat_message_attachments` |
| 权限控制 | `roles` + `permissions` + `user_roles` + `role_permissions` |
| 模型训练 | `training_tasks` → `training_metrics` → `model_versions` |
| 多 Agent | `agent_registry` + `agent_invocations` + `agent_teams`（预留） |
| 操作审计 | `operation_logs` |

## 六、实施路线图

```
✅ 第 1 周（P0 - 已完成）
  ├── users 表加 user_type + patient_profiles + doctor_patient_relations
  ├── medical_records + 注册页类型选择 + 自动建档案
  ├── 侧边栏差异化 + 检测关联患者
  ├── 患者管理页 + 病例管理页
  └── 检测时关联 patient_profile_id

✅ 第 2 周（P1 - 已完成）
  ├── LLM 病史感知分析（检测 + 历史病例 → AI 综合）
  ├── HistoryPage 前端（列表 + 详情 + 日期筛选）
  ├── DashboardPage 统计看板（ECharts 图表）
  ├── Chat 检测走完整管道（入库 + AI 分析）
  ├── 快捷检测自动入库
  └── 对话上下文绑定患者（医生下拉选患者）

⏳ 第 3 周（P1 收尾 + P2 起步）
  ├── 检测报告生成 + PDF 导出
  ├── cxr_images 自动记录 + 病例附件上传
  ├── 操作日志完善
  └── P2 准备工作（RAG 医学知识库调研）
```
