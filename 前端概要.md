# ChestVision 胸片X光智能分析系统 — 前端概要

> 基于 Vue 3 + Element Plus 的医疗影像 AI 辅助诊断平台前端实现

---

## 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3（Composition API） | `<script setup>` 语法，TypeScript 未启用 |
| 构建 | Vite 8 | 极速 HMR 开发体验 |
| UI 库 | Element Plus 2.14 | 企业级组件库，全局主题覆盖为医疗青绿色 |
| 状态管理 | Pinia 3 | `userStore` + `agentStore` |
| 路由 | Vue Router 4 | History 模式，路由守卫鉴权 |
| HTTP | Axios | 统一拦截器处理 Token/错误 |
| 图表 | ECharts 6 | 数据看板可视化 |
| Markdown | markdown-it | AI 对话回复渲染 |
| 样式 | SCSS | 主题变量 + 全局覆盖 Element Plus |
| 流式 | Fetch SSE | 多 Agent 流式对话 |

## 页面清单（10 个页面）

### 1. 登录/注册 (`/login`) — `LoginPage.vue`
- 双面板滑动切换（登录 ↔ 注册）
- 登录：用户名 + 密码 + 记住我 + 忘记密码
- 注册：姓名 + 邮箱 + 邮箱验证码（60s倒计时）+ 用户类型选择 + 密码
- 左上角校徽 logo + 品牌展示区
- 毛玻璃卡片 + 背景光晕，视觉层次丰富

### 2. 智能对话 (`/chat`) — `ChatPage.vue` ⭐核心
- **欢迎模式**（无对话时）：
  - 居中肺部 logo + "你好，{用户名}" + "开启智能医疗问答"
  - 3个快捷提问卡片 + 快捷检测按钮（单图/批量）
  - 患者选择器（医生可见）+ 文件上传 + 文本输入
  - 侧边栏可展开查看历史对话
- **对话模式**（有消息后）：
  - 左右气泡布局：用户青色渐变气泡（右）+ AI 白色气泡（左，带🫁头像）
  - Markdown 渲染 + 打字动画 + 图片附件预览
  - **Multi-Agent 协作流程可视化**：显示 Supervisor→Detection→Diagnosis→Report 节点链路
  - **工具调用可视化**：展示具体工具名和状态
  - **知识库来源展示**：标注 RAG 检索结果
  - 检测结果卡片内嵌 + PDF 报告下载
- **SSE 流式交互**：实时接收 Agent 节点事件/思考文本/最终回复
- 左侧会话列表面板：新建/切换/删除历史对话

### 3. 检测工作台 (`/detection`) — `DetectionPage.vue`
- 左右分栏：左侧上传 + 右侧结果
- 拖拽/点击上传胸片（支持 JPG/PNG/BMP/TIFF）
- 置信度滑块（0.1-0.9）+ 模型选择（显示各模型 mAP50 指标）
- 检测结果：统计卡片（病灶数/位置/置信度） + 标注图渲染 + 病灶列表
- 检测完成后自动弹出 AI 医生推荐弹窗

### 4. 数据看板 (`/dashboard`) — `DashboardPage.vue`
- 4 个 KPI 卡片：检测总次数 / 病灶总数 / 平均耗时 / 高风险案例
- ECharts 图表：检测趋势折线图 / 病灶类型饼图 / 风险等级分布
- 医生工作量统计表（管理者视角）

### 5. 历史记录 (`/history`) — `HistoryPage.vue`
- 时间线设计（非传统表格），每条记录带状态圆点
- 日期范围筛选 + 分页
- 点击展开详情弹窗（病灶标签 + 推理耗时 + 缩略图）

### 6. 患者管理 (`/patients`) — `PatientManagePage.vue`
- 患者列表表格 + AI 医生推荐待确认区域（管理员）
- 管理员：分配医生弹窗、移除关联
- 医生：查看患者病例跳转

### 7. 病例管理 (`/medical-records`) — `MedicalRecordPage.vue`
- 患者筛选下拉 + 病例表格（就诊类型/主诉/状态/日期）
- 新建/查看/编辑/打印/删除操作
- 附件上传支持

### 8. 模型训练 (`/training`) — `TrainingPage.vue`
- 训练任务列表（进度条 + 状态标签 + Epoch 进度）
- 实时监控面板：损失曲线 + mAP 曲线（ECharts）
- 模型版本管理：全局默认模型切换、验证评估、导出/下载权重
- 数据集上传（ZIP）

### 9. 个人中心 (`/profile`) — `ProfilePage.vue`
- 基本信息编辑（用户名/邮箱/手机）
- 修改密码（旧密码 + 新密码 + 确认）
- 患者健康档案（仅患者角色）：姓名/年龄/性别/血型/身高体重/过敏史/紧急联系人

### 10. 404 页面
- 路由兜底，未匹配路径重定向到登录页

## 特色技术实现

1. **SSE 流式对话**：基于 Fetch API + ReadableStream，支持中断（AbortController）
2. **Multi-Agent 流程可视化**：实时展示 6 个 Agent 节点的协作状态
3. **快捷检测模式**：跳过对话直接调用 YOLO 检测，结果嵌入对话卡片
4. **PDF 报告生成**：服务端生成真实 PDF，前端 fetch blob 下载
5. **邮箱验证码注册**：60s 倒计时 + 6 位验证码防刷
6. **全局错误监控**：Vue + JS + Promise 三层错误捕获，本地存储日志
7. **JWT 自动续期**：Axios 拦截器自动注入 Token，401 自动跳转登录
