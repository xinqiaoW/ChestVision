**关键关系说明：**

| 关系 | 说明 |
|------|------|
| `users` 1:N `chat_sessions` | 一个用户可创建多个对话会话 |
| `chat_sessions` 1:N `chat_messages` | 一个会话包含多条消息 |
| `chat_messages` 1:N `chat_message_attachments` | 一条消息可附带多个附件（图片/卡片） |
| `users` 1:N `detection_tasks` | 一个用户可发起多次检测 |
| `detection_tasks` 1:N `detection_results` | 一次检测可发现多个病灶 |
| `detection_tasks` 1:1 `cxr_images` | 一次检测任务对应一张胸片 |
| `patients` 1:N `cxr_images` | 一个患者可有多张胸片 |
| `patients` 1:N `detection_tasks` | 一个患者可有多次检测记录 |
| `detection_scenes` 1:N `detection_tasks` | 胸片场景下的所有检测 |
| `detection_scenes` 1:N `training_tasks` | 胸片场景下的训练任务 |
| `detection_scenes` 1:N `model_versions` | 胸片场景下的模型版本 |
| `model_versions` 1:N `detection_tasks` | 记录每次检测用的模型版本 |
| `users` 1:N `operation_logs` | 用户操作审计 |
| `agent_registry` 1:N `agent_invocations` | 记录每次 Agent 调用 |

## 三、模块一：用户与权限（RBAC）— 5 张表

> 状态：✅ **已有，无需改动**

### 3.1 `users` — 用户表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | INTEGER | PK, AUTO_INCREMENT | 用户 ID |
| `username` | VARCHAR(50) | UNIQUE, NOT NULL, INDEX | 用户名 |
| `email` | VARCHAR(100) | UNIQUE, NOT NULL, INDEX | 邮箱 |
| `hashed_password` | VARCHAR(255) | NOT NULL | bcrypt 加密密码 |
| `phone` | VARCHAR(20) | NULLABLE | 手机号 |
| `avatar` | VARCHAR(500) | NULLABLE | 头像 URL |
| `is_active` | BOOLEAN | DEFAULT TRUE | 是否启用 |
| `is_superuser` | BOOLEAN | DEFAULT FALSE | 是否超级管理员 |
| `last_login_at` | DATETIME | NULLABLE | 最后登录时间 |
| `created_at` | DATETIME | DEFAULT NOW() | 创建时间 |
| `updated_at` | DATETIME | DEFAULT NOW(), ON UPDATE NOW() | 更新时间 |

### 3.2 `roles` — 角色表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | INTEGER | PK, AUTO_INCREMENT | 角色 ID |
| `name` | VARCHAR(50) | UNIQUE, NOT NULL | 角色标识：admin/operator/viewer/radiologist/... |
| `display_name` | VARCHAR(100) | NOT NULL | 角色显示名：管理员/放射科医生 |
| `description` | VARCHAR(500) | NULLABLE | 角色描述 |
| `is_system` | BOOLEAN | DEFAULT FALSE | 是否系统内置（不可删除） |
| `created_at` | DATETIME | DEFAULT NOW() | 创建时间 |

### 3.3 `permissions` — 权限表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | INTEGER | PK, AUTO_INCREMENT | 权限 ID |
| `code` | VARCHAR(100) | UNIQUE, NOT NULL | 权限编码：`detection:task:create` |
| `name` | VARCHAR(100) | NOT NULL | 权限名称 |
| `module` | VARCHAR(50) | NOT NULL | 所属模块：auth/detection/training/agent/system |
| `description` | VARCHAR(500) | NULLABLE | 权限描述 |

### 3.4 `user_roles` — 用户-角色关联表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | INTEGER | PK, AUTO_INCREMENT | 关联 ID |
| `user_id` | INTEGER | FK → users.id, NOT NULL, INDEX | 用户 ID |
| `role_id` | INTEGER | FK → roles.id, NOT NULL, INDEX | 角色 ID |
| `created_at` | DATETIME | DEFAULT NOW() | 创建时间 |

### 3.5 `role_permissions` — 角色-权限关联表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | INTEGER | PK, AUTO_INCREMENT | 关联 ID |
| `role_id` | INTEGER | FK → roles.id, NOT NULL, INDEX | 角色 ID |
| `permission_id` | INTEGER | FK → permissions.id, NOT NULL, INDEX | 权限 ID |
